#  Online Bookstore Sales & Customer Analytics

##  Project Overview

This project analyzes an online bookstore dataset to uncover meaningful insights into sales performance, customer behavior, and inventory trends.
The goal is to transform raw data into actionable business insights using SQL, Python, and Power BI.

---

## Tools & Technologies

* SQL (MySQL) → Data extraction & transformation
* Python (Pandas, Matplotlib) → Data cleaning & analysis
* Power BI → Interactive dashboard & visualization

---

---

##  Key Insights

* Fiction genre generates the highest revenue
* Top 10% of customers contribute to the majority of sales
* Sales peak during November (seasonal trend)
* Certain books show low inventory and risk stockouts

---

##  Dashboard Features

* KPI Metrics: Total Revenue, Total Orders, Average Order Value
* Sales trend analysis over time
* Customer segmentation (top vs regular customers)
* Product and inventory performance insights

---

##  How to Run the Project

### SQL Setup

* Import dataset into MySQL
* Run `schema.sql` to create tables
* Execute `queries.sql` for analysis

### Python Analysis

* Open `analysis.ipynb` in Jupyter Notebook
* Run all cells for data cleaning and exploration

### Power BI Dashboard

* Open `dashboard.pbix` in Power BI
* Refresh data if needed
